# Job_Search

This repository provides users with the ability to add business cards, filter existing cards, and delete them. The main purpose is to reinforce the topic of the React Toolkit.

# Features

- Redux Toolkit
- React Router
- Redux
- API Calls
- React Hooks
- Conditional Rendering
- Props
- Map
- UUID Creation
- Slice Configuration
